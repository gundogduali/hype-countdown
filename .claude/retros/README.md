# Retros

Every team agent writes a retro file here before finishing its work (enforced by the SubagentStop hook).

File name: `<agent-name>-<YYYY-MM-DD-HHMMSS>.md`

Content template:

```markdown
# Retro — <agent> — <task summary>
## What I did
## What I found in self-review
## What I fixed
## What the orchestrator needs to know
```
